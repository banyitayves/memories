## 📊 API RESPONSE EXAMPLES

### Search Books
```
GET /api/resources/search?q=gatsby
→ { results: [{id, title, author, type, available, location}] }
```

### Get Account
```
GET /api/resources/account
→ { account: {name, studentId, email, currentLoans, fineAmount} }
```

### Get Study Rooms
```
GET /api/spaces/rooms
→ { rooms: [{id, name, capacity, amenities, availability}] }
```

### Get Occupancy
```
GET /api/spaces/occupancy
→ { floors: [{floor, occupancyLevel, quietSpots, crowdedAreas}] }
```
