## 📖 THE 12 FEATURES

### Resource Management (3)
1. **📚 One-Search Catalog** - Find 50,000+ books, e-books, journals
2. **👤 Account Portal** - View loans, renew books, track reading
3. **📦 Book Holds & Alerts** - Reserve books, get pickup notifications

### Space Coordination (3)
4. **🏢 Study Room Booking** - Instant reservations with visual calendar
5. **🏢 Occupancy Tracker** - Real-time crowding levels per floor
6. **🔌 Equipment Reservations** - Borrow laptops, cameras, chargers

### Learning Support (3)
7. **🎓 Research Guides** - Subject-specific (Biology, Literature, Business)
8. **💬 Live Librarian Chat** - Instant Q&A and research help
9. **📚 Citation Generator** - APA, MLA, Chicago format citations

### Social Features (3)
10. **✨ New Books Shelf** - Trending and newly arrived titles
11. **💬 Discussion Forums** - Study groups and academic discussions
12. **🗺️ Interactive Map** - Floor plans with call numbers

```bash

```

### 2️⃣ Run Locally
```bash

```
Open: **http://localhost:3000**

### 3️⃣ Deploy to Vercel
```bash

```
