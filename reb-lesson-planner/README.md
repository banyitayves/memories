## ğŸš€ How to Use

### Getting Started
1. Open `index.html` in any web browser
2. Start from the "Basic Information" section
3. Fill in the required fields (marked with *)

### Creating a Lesson Plan

#### 1. Basic Information (Section 1)
- Enter **Subject** (Mathematics, English, Biology, etc.)
- Select **Grade Level**
- Specify **Unit/Module** and **Topic**
- Set number of lessons and duration per lesson
- Add teacher name and school (optional)
- Write general objectives for the unit

#### 2. Learning Outcomes (Section 2)
- Add specific learning outcomes
- Each outcome should start with an action verb
- Click "+ Add Learning Outcome" to add more
- Examples:
  - "Identify and classify different types of fractions"
  - "Apply mathematical principles to solve real-world problems"
  - "Analyze historical events and their impact"

#### 3. Teaching Activities (Section 3)
- **Introduction/Motivation**: Hook students' attention and connect to prior knowledge
- **Main Activities**: Add multiple activities with timeframes
- **Conclusion/Consolidation**: Summarize and consolidate learning

#### 4. Assessment (Section 4)
- Describe **Formative Assessment** (during lesson)
- Describe **Summative Assessment** (end of unit)
- Select assessment methods used:
  - Observation
  - Questioning
  - Written Tests
  - Practical Activities
  - Projects
  - Group Work

#### 5. Resources (Section 5)
- List all teaching/learning materials needed
- Add curriculum references
- Include special notes for learners with different needs

#### 6. Preview (Section 6)
- View your completed lesson plan
- See how it will look when printed
