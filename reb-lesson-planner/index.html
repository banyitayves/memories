🔑 Configure AI API Key

To use AI-powered lesson generation, provide an API key from one of these services:

OpenAI API

  1. Go to platform.openai.com/api-keys
  2. Create a new API key or copy existing one
  3. Paste it below

⚠️ Never share your API key. It's stored locally in your browser only.

Anthropic Claude API

  1. Go to console.anthropic.com
  2. Create a new API key
  3. Paste it below

⚠️ Never share your API key. It's stored locally in your browser only.

Demo Mode

Use Demo Mode to generate sample lesson plans without an API key. Perfect for testing!
