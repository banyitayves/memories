# Quick Reference: Phase 2 Features

## 🚀 Quick Start (60 Seconds)

### 1️⃣ Generate & Save a Lesson (Auto-saves!)
```
1. Click "AI Generate" tab
2. Select Grade & Subject
3. Fill lesson title, duration, competence, objective
4. Click "Generate Lesson Plan with AI"
5. ✅ Lesson auto-saves to "My Lessons"!
```

### 2️⃣ Access Your Saved Lessons
```
1. Click "My Lessons" tab
2. See all your saved lessons (newest first)
3. Click "📖 Open" to load lesson
4. Click "⬇️ Download" to get PDF/Word
5. Click "🗑️ Delete" to remove
```

### 3️⃣ Change Language (Entire System!)
```
1. Find language dropdown in top header
2. Select: English | Kiswahili | Français | Kinyarwanda  
3. Everything changes to that language!
4. Your language choice is saved automatically
```

## 💡 Tips & Tricks

### Tip 1: Your Lessons Stay in Browser
- All lessons saved on THIS computer/device
- Open same browser anytime → lessons still there
- Different browser/computer = no lessons (need backup)

### Tip 2: Share Lessons Easily
```
Generate → My Lessons → Download → Email/Print/Share
```

### Tip 3: Switch Languages Anytime
- Mid-work? Can switch language
- Entire system updates immediately
- Continue editing in new language

### Tip 4: Keep Backup
- Sometimes browsers clear storage
- Download important lessons as PDF
- Store PDFs in cloud (Google Drive, OneDrive)

### Tip 5: Editing Saved Lessons
```
1. Go to My Lessons
2. Click "📖 Open" on lesson
3. Edit form fields as needed
4. Click "Save Plan" (manual override)
5. Updated lesson saved
```

## ❓ FAQ

**Q: Do I need to click "Save" after generating?**
A: NO! Lessons auto-save automatically. You'll see confirmation message!

**Q: Can I access lessons on different devices?**
A: Not automatically. But you can download as PDF and email to yourself, or use cloud storage.

**Q: What if I clear my browser cache?**
A: Lessons will be deleted. Always download important lessons as PDF!

**Q: How many lessons can I save?**
A: Unlimited! System keeps last 50 in active index, but you can save way more.

**Q: Can I teach in English but save in Kiswahili?**
A: Yes! Generate in English, then switch language and download - PDF saves in current language.

**Q: What happens if I close the browser?**
A: Lessons stay on this computer! Next time you open, they're still in "My Lessons".

**Q: Can I delete a lesson?**
A: Yes! Open My Lessons → Click 🗑️ Delete on the lesson card.

**Q: Is my data sent to internet?**
A: NO! Everything saves on your computer. No cloud unless you add it.

## 🆘 Troubleshooting

### Lesson not saving?
1. Check if localStorage is enabled (Settings → Privacy)
2. Check browser has space (not full)
3. Try refreshing page
4. Try different browser

### Language not changing?
1. Make sure dropdown is visible
2. Try selecting different language twice
3. Refresh page and try again

### Lesson not loading?
1. Check lesson exists in My Lessons
2. Try refreshing page
3. Clear browser cache
4. Try different lesson first

### Can't download?
1. Make sure PDF viewer installed
2. Check downloads folder
3. Try Word format instead
4. Try different browser

## 🎓 Best Practices

### For Creating Lessons
✅ DO:
- Use clear, simple lesson titles
- Fill in ALL required fields (marked with *)
- Provide full Key Competence from REB syllabus
- Write specific, measurable learning objectives
- Include special needs considerations

❌ DON'T:
- Leave duration blank (needs valid time)
- Skip competence field (system needs this)
- Put same subject/grade/title multiple times

### For Managing Lessons
✅ DO:
- Download important lessons as PDF backup
- Organize lessons with clear titles
- Review generated lesson and edit if needed
- Keep browser storage clean (delete old versions)

❌ DON'T:
- Rely only on browser storage for backup
- Ignore generated lesson preview
- Assume all lessons stay forever (can be deleted)

### For Language Use
✅ DO:
- Use language that teachers prefer
- Keep system language consistent per session
- Download in language used for teaching

❌ DON'T:
- Mix languages within single lesson
- Change language constantly (confusing)
- Expect AI prompts to be in different language than selected
