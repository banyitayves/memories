## 🎬 GO! Follow These 8 Steps:

### STEP 1: Open System (1 min)
```
Browser → http://localhost:5500/reb-lesson-planner/index.html
```

### STEP 2: Login (30 secs)
```
Click "👁️ Demo Access" tab
Click "🚀 Continue as Demo User"
You're in! ✅
```

### STEP 3: Navigate to AI Generate (30 secs)
```
LEFT SIDEBAR → Click "🤖 AI Generate" or similar
Or find "AI Generate" tab at top
```

### STEP 4: Fill Lesson Form (2 mins)
```
Copy & paste these values:

Education Level:  Primary (P1-P6)
Grade:           P3
Subject:         Mathematics
Lesson Title:    Introduction to Fractions
Duration:        45 minutes
Key Competence:  "The learner can use fractional numbers to describe 
                  and compare quantities in real-life situations."
Learning Obj:    "Students will identify and create simple fractions 
                  (1/2, 1/4) using real objects and drawings."
Checkboxes:      ☑ Activities  ☑ Assessment  ☑ Resources  ☑ Reflection
```

### STEP 5: Generate (3 mins)
```
Click: "🤖 Generate Lesson Plan with AI"

Wait... (it will take 2-3 seconds)

See: ✅ "Lesson plan generated and saved!"

PLUS: Your lesson preview appears! 📋
```

### STEP 6: Download as PDF (1 min)
```
Below the preview, find: [📄 PDF] [📘 Word]

Click [📄 PDF]

Wait 1 second...

✅ PDF downloads to your Downloads folder!
```

### STEP 7: Find & Open PDF (1 min)
```
File Explorer → Downloads folder

Look for: "REB_Lesson_Plan_*"

Double-click → Opens in PDF viewer ✅

YOUR LESSON PLAN IS READY! 🎉
```

### STEP 8: Try Kiswahili (1 min) - BONUS!
```
TOP RIGHT → Find 🌐 Language dropdown

Click → Select "Kiswahili"

Watch ENTIRE system change to Kiswahili! ✨

Now generate lesson again - it's in Kiswahili!
Download PDF - labels are in Kiswahili! 🇹🇿
```

| Problem | Fix |

| Language not changing | Refresh page (F5), then try dropdown again |

| Generation takes too long | Wait 5 seconds max, then refresh |
